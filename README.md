# ParallelCodes
Sample codes for a course on parallel programming and general-purpose GPU programming

## Content
The CUDA folder contains the following examples of CUDA C programming:
a) Simple examples of vector addition and square matrix addition.
b) Use of pinned, mapped and unified memory.
c) Sum of elements of a real-valued vector using several reduction
kernels.
d) Floyd-Warshall algorithm for computing the shortest paths in a
dense graph.
e) Numerical solution of the 1D linear advection equation using finite
differences.
